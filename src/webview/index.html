Vim Mode
Vim Commands:
  • j/k - Move down/up within current section
  • h/l - Switch between form (h) and saved requests (l)
  • Tab - Focus next field
  • Shift+Tab - Focus previous field
  • i - Enter insert mode (edit field)
  • Esc - Exit insert mode
  • Enter - Send request (form) / Load request (saved)
  • s - Save current request
  • r - Refresh saved requests
  • d - Delete selected saved request
  • ? - Toggle this help
